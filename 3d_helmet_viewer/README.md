# 3D Helmet Viewer 🤖⛑️

Автономний 3D переглядач шолома з темною атмосферою та адаптивним освітленням.

## 📁 Склад папки

```
3d_helmet_viewer/
├── 3d_viewer.html              # Головний HTML файл з 3D viewer
├── DamagedHelmet.glb           # 3D модель пошкодженого шолома
├── start_3d_viewer.py          # HTTP сервер для запуску
├── night_lighting_control.py   # Адаптивне управління освітленням
└── README.md                   # Цей файл з інструкціями
```

## 🚀 Швидкий запуск

### Метод 1: Простий запуск
```bash
cd 3d_helmet_viewer
python start_3d_viewer.py
```
Потім відкрийте: `http://localhost:8080/3d_viewer.html`

### Метод 2: З адаптивним освітленням
```bash
# Термінал 1: Запуск сервера
cd 3d_helmet_viewer
python start_3d_viewer.py

# Термінал 2: Запуск контролера освітлення
cd 3d_helmet_viewer
python night_lighting_control.py
```

## ⚙️ Особливості

### 🎨 Візуальні ефекти:
- **Чорний фон** для драматичної атмосфери
- **Темне освітлення** (`exposure="0.15"`)
- **М'які тіні** (`shadow-intensity="0.05"`)
- **Автоматичне обертання** моделі
- **Інтерактивне управління** камерою

### 🔦 Адаптивне освітлення (опціонально):

## 📝 Налаштування

### Зміна моделі:
Замініть `DamagedHelmet.glb` на іншу GLB/GLTF модель та оновіть `src` в HTML файлі.

### Налаштування освітлення:
Відредагуйте параметри в `3d_viewer.html`:
```html
exposure="0.15"           # Яскравість (0.01-2.0)
shadow-intensity="0.05"   # Інтенсивність тіней (0.0-1.0)
metalness="0.1"          # Металічність (0.0-1.0)
roughness="0.95"         # Шорсткість (0.0-1.0)
```

### Зміна порту:
Відредагуйте `start_3d_viewer.py`, змініть `port=8080` на потрібний.

## 🎮 Управління

- **Обертання**: Перетягування лівою кнопкою миші
- **Зум**: Колесо миші або pinch на тачскрині
- **Панорамування**: Права кнопка миші + перетягування
- **Автообертання**: Вимикається при взаємодії

## 🔧 Усунення проблем

### Модель не завантажується:
- Перевірте, чи файл GLB існує у папці
- Переконайтеся, що сервер запущено
- Перевірте консоль браузера на помилки

### Темний екран:
- Збільште `exposure` в HTML файлі
- Перевірте, чи модель має валідні матеріали

### Playwright помилки:
```bash
pip install playwright
playwright install chromium
```

## 🎯 Готові налаштування

- **Помірно темний**: `exposure="0.15"` (поточний)
- **Збалансований**: `exposure="0.5"`
- **Яскравий**: `exposure="1.0"`
