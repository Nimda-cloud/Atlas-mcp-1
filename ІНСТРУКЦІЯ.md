# 🤖 Atlas Autonomous System - Українська інструкція

Це автономна система штучного інтелекту для керування macOS та інших платформ.

## 🚀 Швидкий старт

### Автоматичне встановлення (рекомендовано)

1. **Завантажити та запустити скрипт встановлення:**
   ```bash
   curl -fsSL https://raw.githubusercontent.com/oleg121203/Atlas-mcp/main/install_macos.sh | bash
   ```

2. **Запустити Atlas:**
   ```bash
   cd ~/Atlas
   ./start_atlas.sh
   ```

3. **Відкрити веб-інтерфейс:**
   Перейти на [http://localhost:8000](http://localhost:8000) у браузері

### Ручне встановлення

1. **Встановити залежності:**
   ```bash
   # Встановити Homebrew (якщо ще не встановлено)
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   
   # Встановити Python 3.11+
   brew install python@3.11
   
   # Встановити Ollama
   brew install ollama
   ```

2. **Клонувати репозиторій:**
   ```bash
   git clone https://github.com/oleg121203/Atlas-mcp.git
   cd Atlas-mcp
   ```

3. **Встановити Python залежності:**
   ```bash
   python3 -m venv atlas_env
   source atlas_env/bin/activate
   pip install -r requirements.txt
   ```

4. **Запустити Atlas:**
   ```bash
   ./start_atlas.sh --local
   ```

## 🔧 Конфігурація

### Змінні середовища

Створіть файл `.env` з необхідними налаштуваннями:

```env
# LLM конфігурація
ATLAS_LLM1_MODEL=llama3.1:8b
ATLAS_LLM1_PROVIDER=ollama
OLLAMA_URL=http://localhost:11434

# Веб-інтерфейс
ATLAS_WEB_PORT=8000
ATLAS_DEBUG=false

# Дані
ATLAS_DATA_DIR=./data
```

### Автоматизація

Система підтримує автоматичне виконання завдань через:

- **AppleScript** - для автоматизації macOS
- **Веб-автоматизація** - через Playwright
- **Голосовий синтез** - для аудіо відповідей
- **Українська мова** - природна обробка команд

## 🚨 Усунення проблем

### Типові помилки

1. **Ollama не запущений:**
   ```bash
   brew services start ollama
   ollama pull llama3.1:8b
   ```

2. **Порт зайнятий:**
   ```bash
   lsof -ti:8000
   kill -9 $(lsof -ti:8000)
   ```

3. **Проблеми з залежностями:**
   ```bash
   pip install --upgrade pip
   pip install -r requirements.txt
   ```
