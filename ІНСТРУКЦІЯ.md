# 🤖 Інструкція по використанню Atlas Autonomous System

**Система автономного управління комп'ютером для macOS**

## 📖 Огляд системи

Atlas - це передова автономна система, яка використовує множинних AI-агентів для інтелектуального управління та автоматизації вашого macOS комп'ютера. Система побудована з локальною обробкою LLM, модульною архітектурою та розширеними можливостями автоматизації.

### 🎯 Основні можливості

- **Автономне управління**: Система може самостійно приймати рішення та виконувати задачі
- **Три AI-агенти**: Спеціалізовані агенти для різних функцій
- **Локальна обробка**: Вся AI-обробка відбувається локально для забезпечення приватності
- **Голосова взаємодія**: Підтримка команд голосом та озвучування відповідей
- **Веб-інтерфейс**: Зручний веб-інтерфейс для керування системою
- **Нативна інтеграція macOS**: Повна інтеграція з системними функціями macOS

## 🚀 Швидкий старт

### Варіант 1: Автоматичне встановлення (Рекомендовано)

1. **Завантажте та запустіть скрипт встановлення:**
   ```bash
   curl -fsSL https://raw.githubusercontent.com/oleg121203/Atlas-mcp/main/install_macos.sh | bash
   ```

2. **Запустіть Atlas:**
   ```bash
   cd ~/Atlas
   ./start_atlas.sh
   ```

3. **Відкрийте веб-інтерфейс:**
   Перейдіть за адресою [http://localhost:8000](http://localhost:8000) у вашому браузері

### Варіант 2: Використання Docker

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/oleg121203/Atlas-mcp.git
   cd Atlas-mcp
   ```

2. **Запустіть за допомогою Docker:**
   ```bash
   docker-compose up -d
   ```

3. **Відкрийте інтерфейс:**
   [http://localhost:8000](http://localhost:8000)

### Архітектура системи

#### Три основні агенти:

1. **LLM1 (Інтерфейс-агент)**
   - Обробляє взаємодію з користувачем
   - Управляє пам'яттю розмов
   - Формалізує запити користувача

2. **LLM2 (Оркестратор)**
   - Планує та координує виконання задач
   - Розподіляє роботу між інструментами
   - Управляє робочими процесами

3. **LLM3 (Монітор)**
   - Відстежує безпеку системи
   - Аналізує аномалії
   - Автоматично реагує на проблеми

#### Модульна архітектура MCP:

Система використовує модульні MCP (Model Context Protocol) сервери:

- **Automation MCP**: Загальна автоматизація
- **macOS Automator MCP**: Нативні функції macOS
- **TTS MCP**: Синтез мовлення
- **Browser MCP**: Автоматизація браузера

## ⚙️ Налаштування

### Змінні середовища

```bash
# Основна конфігурація
export ATLAS_DATA_DIR="/path/to/data"
export ATLAS_LOG_LEVEL="INFO"
export ATLAS_WEB_PORT="8000"

# Конфігурація LLM
export ATLAS_LLM_PROVIDER="ollama"
export ATLAS_LLM_MODEL="llama3.1:8b-instruct"
export OLLAMA_HOST="localhost:11434"

# Конфігурація голосу
export ATLAS_TTS_AGENT_VOICE_LLM1="Alex"
export ATLAS_TTS_AGENT_VOICE_LLM2="Samantha"
export ATLAS_TTS_AGENT_VOICE_LLM3="Daniel"
```

### Файли конфігурації

**Основна конфігурація** (`data/config/atlas_config.json`):
```json
{
  "system": {
    "name": "Atlas Autonomous System",
    "version": "1.0.0",
    "data_dir": "./data",
    "log_level": "INFO"
  },
  "automation": {
    "enabled": true,
    "safe_mode": true,
    "require_confirmation": true
  },
  "voice": {
    "tts_enabled": true,
    "stt_enabled": true,
    "voice_commands": true
  }
}
```

## 🔧 Розширені можливості

### Користувацькі скрипти автоматизації

Створіть власні скрипти в `data/scripts/`:

```python
# custom_automation.py
import asyncio
from atlas_core import AtlasCore

async def custom_task():
    atlas = AtlasCore()
    
    # Відкрити TextEdit
    await atlas.macos_automation.manage_applications("open", "TextEdit")
    
    # Ваша логіка тут
    print("Виконую користувацьку задачу...")
```

### Інтеграція з Shortcuts

Atlas може запускати користувацькі Shortcuts:

```python
# Через веб-інтерфейс або API
{
  "action": "shortcuts",
  "shortcut_name": "My Automation",
  "input_text": "Привіт Atlas!"
}
```

### Додавання власних MCP серверів

1. **Створіть MCP сервер:**
   ```python
   from mcp_automation_server import MCPTool, MCPResponse
   
   class CustomMCP:
       def __init__(self):
           self.tools = {
               "custom_action": MCPTool(
                   name="custom_action",
                   description="Моя користувацька дія",
                   parameters={...}
               )
           }
   ```

2. **Зареєструйте в конфігурації:**
   ```bash
   export ATLAS_MCP_SERVERS="automation,macos-automator,tts,custom"
   export ATLAS_MCP_CUSTOM_URL="http://localhost:4005"
   ```

## 🚦 Керування системою

### Запуск Atlas
```bash
# Автоматичний режим
./start_atlas.sh

# Docker режим
./start_atlas.sh --docker

# Локальний режим
./start_atlas.sh --local

# Фоновий режим
./start_atlas.sh --background
```

### Зупинка Atlas
```bash
# Якщо працює через Docker
docker-compose down

# Якщо працює локально
kill $(cat data/atlas.pid)

# Або натисніть Ctrl+C якщо працює в терміналі
```

### Перезапуск Atlas
```bash
# Повний перезапуск
./start_atlas.sh --docker
docker-compose restart

# Або для локального режиму
kill $(cat data/atlas.pid)
./start_atlas.sh --local
```
